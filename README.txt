Приложение состоит из 2 исполняемых файлов: 
- Клиентское: client.py
- Серверное: map_server.py

Для Windows рекомендуется для запуска использовать venv, для этого необходимо:
    1) Перейти в папку venv/Scripts
    2) Запустить файл activate.bat
    3) Запустить Приложение

Клиент запускается из консоли без параметров. При работе
