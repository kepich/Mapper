Приложение состоит из 2 исполняемых файлов: 
- Клиентское: client.py
    - Для запуска необходимы также: server.py, utils.py
- Серверное: map_server.py
    - Для запуска необходимы также: server.py, utils.py, db.py

Для Windows рекомендуется для запуска использовать venv, для этого необходимо:
    1) Перейти в папку venv/Scripts
    2) Запустить файл activate.bat
    3) Запустить Приложение

    Клиент запускается из консоли без параметров. При работе необходимо ввести строку из 8 символов, соответствующих
окружению. Далее, если сервер имеет такую ситуацию, то присылается ответ, содержащий путь, иначе проиходит корректировка
окружения. Перед запуском необходимо скорректировать адрес сервера в main, а именно переменные server_ip и server_port

    Сервер при запуске имеет один необязательный параметр: \t - указывается для того, чтобы сгенерировать новую базу
данных и удалить старую. Для того, чтобы использовать уже созданную базу данных сервер запускается без параметров. Выход
из рабочего цикла производится нажатием Ctrl-C

    Файл myDb.db содержит ~400000 записей, поэтому рекомендуется не создавать новый, так как генерация занимает значительное
время.
